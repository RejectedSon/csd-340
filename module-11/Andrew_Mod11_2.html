Assignment 11.2: JavaScript - Part 3


This webpage will allow the user to input items into an array, delete the latest entry into the array, as well as display the entire array.

  • To add an item: Input a string into the text box, then click the "ADD" button. A minimum of 3 characters is required to add an item!
  • To remove the last item, click the "REMOVE" button.
  • To display the list, click the "SHOW ARRAY" button.
  • Last but not least, just have fun with it!
